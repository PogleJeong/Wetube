Youtube clone
------

## 1. 프로젝트 소개

Youtube 와 같은 동영상 플랫폼을 클론하였습니다.

해당 프로젝트는 풀스택으로 진행되었으며, ES6 문법을 사용하였습니다.

# 2. 기술스택

Frontend : JS(ES6), SCSS, Webpack, Pug(view)
Backend: Node.js
Database: MongoDB

# 3. 기타 사용한 툴

Editor: visual studio code
Deployment tool: Heroku,
